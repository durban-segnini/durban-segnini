# Durban Segnini Gallery

This repository contains code for a static website built for Durban Segnini Gallery with [Eleventy (11ty)](https://www.11ty.dev/).
It is deployed with **GitHub Pages** and is structured to support adding a CMS in the future.

Follow the instructions below to set up your local development environment.

### 2. Install Node via [mise](https://mise.jdx.dev/)

This project uses [mise](https://mise.jdx.dev/) with a `mise.toml` file to manage Node at a local directory level.
Run (inside the project directory):

```bash
mise install
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Local Development Server

Run:

```bash
npm run start
```

The site will be served locally (usually at [http://localhost:8080](http://localhost:8080))
